Giangbb Studio: Play of a video through WebRTC - VIEWER

This is an implementation of a video broadcasting application. There are two types of users in this application: 1 peer sending media (let's call it Presenter) and N peers receiving the media of the Presenter (let's call them Viewers). Thus, the Media Pipeline is composed by 1+N interconnected WebRtcEndpoints. To run this demo follow these steps:

  1. Open this page with a browser compliant with WebRTC (Chrome, Firefox).
  2. Click on Viewer button to see the Presenter stream.
  3. Click on Stop to finish the communication. When Presenter clicks on Stop, the rest of Viewers (if any) ends the communication too.
